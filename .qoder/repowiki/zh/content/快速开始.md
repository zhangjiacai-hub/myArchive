## 简介

MyArchive是一个用Rust编写的命令行文件压缩与解压工具。它支持将单个文件或整个目录压缩为ZIP格式，也支持从ZIP文件中提取内容和列出ZIP文件的内容列表。该工具基于现代Rust生态系统构建，提供了简洁易用的命令行界面。

## 环境要求

使用MyArchive需要满足以下环境要求：

### 系统要求
- **操作系统**: 支持Windows、macOS、Linux等主流操作系统
- **Rust版本**: Rust 1.70或更高版本
- **Cargo**: Rust包管理器（随Rust安装包一起提供）

### 运行时依赖
MyArchive在运行时依赖以下核心组件：
- **clap**: 命令行参数解析框架
- **zip**: ZIP文件处理库
- **walkdir**: 目录遍历库
- **anyhow**: 错误处理库

这些依赖项会在编译过程中自动下载和安装。

## 常见使用场景

### 场景一：压缩单个文件

**命令格式**:
```bash
archive compress <源文件路径> [-o <输出文件路径>]
```

**示例**:
```bash
archive compress document.pdf
archive compress image.jpg -o backup.zip
```

**预期输出**:
```
已压缩到: document.pdf.zip
```

### 场景二：压缩整个目录

**命令格式**:
```bash
archive compress <目录路径> [-o <输出文件路径>]
```

**示例**:
```bash
archive compress project/
archive compress docs/ -o projects.zip
```

**预期输出**:
```
已压缩到: project.zip
```

### 场景三：解压文件

**命令格式**:
```bash
archive extract <ZIP文件路径> [-o <输出目录>]
```

**示例**:
```bash
archive extract archive.zip
archive extract backup.zip -o extracted_files
```

**预期输出**:
```
已解压到: archive
```

### 场景四：列出ZIP内容

**命令格式**:
```bash
archive list <ZIP文件路径>
```

**示例**:
```bash
archive list documents.zip
```

**预期输出**:
```
大小       修改时间             文件名
------------------------------------------------------------
1024       2024-01-15 14:30     readme.txt
2048       2024-01-16 09:15     images/photo.jpg
```

## 命令行参数详解

### 主要命令

| 命令 | 描述 | 语法 |
|------|------|------|
| `compress` | 将文件或目录压缩为ZIP文件 | `archive compress <source> [--output OUTPUT]` |
| `extract` | 解压ZIP文件 | `archive extract <source> [--output OUTPUT]` |
| `list` | 列出ZIP文件内容 | `archive list <source>` |

### 参数说明

#### 公共参数
- `-o, --output`: 指定输出文件或目录路径
- `--help`: 显示帮助信息

#### compress命令参数
- `source`: 要压缩的源文件或目录路径（必需）
- `output`: 输出的ZIP文件路径（可选）

#### extract命令参数
- `source`: 要解压的ZIP文件路径（必需）
- `output`: 解压输出目录（可选）

#### list命令参数
- `source`: ZIP文件路径（必需）

### 默认行为

1. **输出文件命名规则**:
   - 如果未指定输出文件，程序会自动在源文件名后添加`.zip`扩展名
   - 目录压缩时，默认输出文件名为目录名加`.zip`

2. **解压默认行为**:
   - 如果未指定输出目录，程序会使用ZIP文件名（去掉`.zip`扩展名）作为输出目录名

## 故障排除

### 常见问题及解决方案

#### 1. 编译错误
**问题**: `rustc`命令找不到
**解决方案**: 
- 确保已正确安装Rust和Cargo
- 在终端中重新加载shell配置文件

**问题**: 依赖下载失败
**解决方案**:
```bash
cargo clean
cargo update
cargo build --release
```

#### 2. 权限错误
**问题**: 无法创建输出文件
**解决方案**:
```bash
# 检查目录权限
ls -la /path/to/output/directory

# 修改目录权限
chmod 755 /path/to/output/directory

# 或者使用sudo（不推荐）
sudo archive compress file.txt -o /root/backup.zip
```

#### 3. 文件不存在错误
**问题**: "源路径不存在"
**解决方案**:
```bash
# 检查文件是否存在
ls -la /path/to/source

# 使用绝对路径
archive compress /absolute/path/to/file.txt

# 确保路径正确
archive compress ./relative/path/to/file.txt
```

#### 4. 内存不足错误
**问题**: 处理大文件时内存不足
**解决方案**:
- 分批压缩大目录
- 使用SSD存储设备
- 关闭其他占用内存的应用程序

#### 5. 权限不足错误
**问题**: 无法读取源文件或写入目标目录
**解决方案**:
```bash
# 检查文件权限
stat /path/to/source/file.txt

# 修改文件权限
chmod 644 /path/to/source/file.txt

# 检查目录权限
stat /path/to/output/directory
chmod 755 /path/to/output/directory
```
